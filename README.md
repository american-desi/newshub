# News-XP Orchestrator

Next-generation news aggregation and presentation engine with real-time data orchestration.

## Tech Stack
- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- React Router v7
- Lucide React (icons)

## Quick Start

```bash
npm install
npm run dev
```

The app opens at http://localhost:5173 by default.
